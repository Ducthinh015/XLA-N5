# XLA - Nhận diện biển báo giao thông

Dự án Nhận diện biển báo giao thông bằng YOLO + FastAPI + React.

## Phân công
- Người 1: `datasets/` – Thu thập và gắn nhãn dữ liệu
- Người 2: `configs/`, `models/` – Config YOLO + checkpoint
- Người 3: `scripts/` – Script inference
- Người 4: `backend/` – API FastAPI
- Người 5: `frontend/` – React + Tailwind
